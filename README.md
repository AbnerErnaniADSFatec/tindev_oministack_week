# Aplicativo TinDev Oministack Week 8.0
### NodeJS version > 8 is required, install global yarn with the following command:
```
# npm install yarn -g
```
### Libs required:
```
$ yarn init -y
$ yarn add express
$ yarn add nodemon -D
$ yarn add mongoose
$ yarn add axios
$ yarn add cors
```
### Run this app with:
```
$ npm install
$ yarn dev
```
